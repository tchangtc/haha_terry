Terry — Cross-Platform Installation Guide

All Platforms (Python)

git clone https://github.com/tchangtc/haha_terry.git && cd haha_terry
pip install -e . || pip install -r requirements.txt
export ANTHROPIC_API_KEY="sk-ant-..."
terry

🪟 Windows

# Use PowerShell or cmd.exe
git clone https://github.com/tchangtc/haha_terry.git
cd haha_terry
pip install -e . || pip install -r requirements.txt
set ANTHROPIC_API_KEY=sk-ant-...
terry

# For ANSI colors in older Windows, run:
pip install colorama

🍎 macOS

git clone https://github.com/tchangtc/haha_terry.git && cd haha_terry
pip install -e . || pip install -r requirements.txt
export ANTHROPIC_API_KEY="sk-ant-..."
terry               # CLI mode
terry webui          # Browser mode
terry desktop        # Desktop tray mode

🐧 Linux

git clone https://github.com/tchangtc/haha_terry.git && cd haha_terry
pip install -e . || pip install -r requirements.txt
export ANTHROPIC_API_KEY="sk-ant-..."
terry
# For desktop tray: pip install pystray pillow

📱 Android / iOS (Mobile)

Open WebUI in your mobile browser, then:

# Start on your computer:
terry webui --host 0.0.0.0

# On your phone, open:
http://YOUR_COMPUTER_IP:8670

# Add to Home Screen for app-like experience

Or use Telegram: pip install httpx and run the Telegram gateway.

🐳 Docker (Any Platform)

docker compose up terry

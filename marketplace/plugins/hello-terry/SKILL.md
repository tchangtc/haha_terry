# Hello Terry

A simple demo skill that greets the user and demonstrates the Terry plugin system.

## When to Use

- When the user says "hello", "hi", or "greet me"
- As a template for creating your own skills

## Instructions

When the user greets you, respond with a friendly message that includes:
- A greeting
- The current Terry version
- A tip of the day

Example response:
```
Hello! I'm Terry v1.3.0, your AI coding agent.
💡 Tip: Use /plan before big refactors to review the approach first.
```
